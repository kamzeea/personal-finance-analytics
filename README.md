# Personal Finance Analytics Platform (HCI Project)

**Overview**

The Personal Finance Analytics Platform is a user-centered web application prototype designed to help individuals track spending, monitor financial habits, and achieve savings goals through intuitive analytics and automation.  
This project was developed as part of **ITEC 312 — Human Computer Interaction**. I served as the **Project Manager**, coordinating design decisions, research direction, usability testing, and team deliverables throughout the development lifecycle.  
The platform focuses on reducing friction in personal finance management by combining automated bank synchronization, real-time analytics visualization, and goal-based financial tracking.

---

**Problem Statement**

Many personal finance applications fail because they are:

- Too complex or data-heavy  
- Dependent on manual entry  
- Visually overwhelming  
- Difficult to maintain long-term engagement  

Users often abandon budgeting tools within weeks due to poor usability and cognitive overload.

---

**Solution**

We designed a friction-free financial analytics experience centered around three primary user tasks:

- **Bank Account Integration**  
  - Secure Plaid-based connection  
  - Automatic transaction syncing  
  - Eliminates manual data entry  

- **Expense Tracking & Categorization**  
  - Real-time categorized spending insights  
  - Automated category suggestions  
  - Dashboard-based analytics visualization  

- **Savings Goal Tracking**  
  - Progress visualization with real-time updates  
  - Behavioral motivation features (progress bars & celebration feedback)

**Design Methodology**

The project followed a **User-Centered Design (UCD)** framework grounded in Human–Computer Interaction principles.

- **Research Phase**  
  - User interviews and surveys  
  - Persona development  
  - Pain-point analysis  
  - Behavioral pattern identification  

- **Design Iteration**  
  - Low-fidelity wireframes  
  - Task-flow modeling  
  - Peer review feedback loops  
  - High-fidelity prototyping in Figma  

- **Usability Validation**  
  - Task-based usability testing  
  - Navigation analysis  
  - Interaction clarity evaluation  
  - Iterative refinement cycles  

---

**Usability Testing Results**

- **Task Success Rate:** 100%  
- **Ease-of-Use Score:** 4.8 / 5  
- **Navigation Clarity:** High  
- **Accessibility Compliance:** WCAG AA  

Users described the interface as:  
> “Simple, clean, and easy to move around.”
